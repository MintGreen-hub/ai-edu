微软亚洲研究院始终关注计算机领域的前沿进展，并以论文分享会的形式为大家带来值得关注的前沿研究，促进计算机各大领域的进步。

本系列论文分享会关注计算机领域的各大顶会，邀请论文作者以线上直播的形式与大家分享并探讨论文的研究问题与研究设计。从2019年开展到现在，已经在NLP,CV,System等众多领域展开了一系列顶级会议论文分享。
